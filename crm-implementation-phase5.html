Shared plugin infrastructure DI PluginInfraModule @Global() Child modules (CustomerDataModule) không thể inject từ parent (PluginsModule). @Global() giải quyết mà không cần circular deps hay forwardRef

ExecutionContext building Factory method trong controller Explicit, stateless — tránh NestJS REQUEST scope overhead và complexity. Context được build per-action call
PoolRegistry cho PoolMetricsCollector Inject từ DalModule (@Global) Metrics phải đo đúng pool đang phục vụ request. Phase 4 tạo pool riêng nên metrics luôn là 0

SandboxService — timeout enforcement
src/plugins/sandbox/sandbox.service.tsTypeScript
@Injectable() export class SandboxService { async execute<T>(fn: () => Promise<T>, timeoutMs = 5_000): Promise<T> { return Promise.race([ fn(), new Promise<never>((_, reject) => setTimeout( () => reject(new GatewayTimeoutException('Plugin execution timeout')), timeoutMs ) ), ]); } } // Controller usage — timeout từ manifest.limits.timeoutMs: const contacts = await sandbox.execute( () => core.listContacts(ctx), core.manifest.limits.timeoutMs // 5000ms default );

Analytics — real aggregate queries
src/plugins/cores/analytics/analytics.core.ts (excerpt)TypeScript
async summary(ctx: IExecutionContext): Promise<AnalyticsSummary> { const [totalRow, activeRow] = await Promise.all([ ctx.db.db('users').count('id as count').first(), ctx.db.db('users').count('id as count').where({ is_active: true }).first(), ]); return { totalContacts, activeContacts, tenantId: ctx.tenantId, generatedAt: new Date().toISOString() }; } async trends(ctx: IExecutionContext): Promise<TrendPoint[]> { // Daily registrations last 30 days — RLS scoped to tenant automatically return ctx.db.db('users') .select(ctx.db.db.raw("DATE(created_at) as date"), ctx.db.db.raw("COUNT(id) as count")) .where('created_at', '>=', ctx.db.db.raw("NOW() - INTERVAL '30 days'")) .groupByRaw('DATE(created_at)') .orderBy('date', 'asc'); }

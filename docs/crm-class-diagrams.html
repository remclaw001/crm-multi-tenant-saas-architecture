Vai tròRole

17 interfaces là hợp đồng trung tâm của toàn bộ kiến trúc. Chúng tạo ranh giới abstraction giữa các layer — plugin (L3) chỉ biết interface, không biết implementation cụ thể ở L4/L5. So với phiên bản 14 interfaces, bổ sung thêm 3 interfaces mới cho Query Builder DIP: ISQLDialect, ITenantStrategy, IMigrationDialect.

17 interfaces are the central contracts of the entire architecture. They create abstraction boundaries between layers — plugins (L3) only know the interface, not the concrete implementation at L4/L5. Compared to the 14-interface version, 3 new interfaces for Query Builder DIP have been added: ISQLDialect, ITenantStrategy, IMigrationDialect.

IExecutionContext — ISP Composition

IExecutionContext là composition của 5 interfaces nhỏ: IDbContext, ICacheContext, IEventContext, IServiceContext, ILogger. Plugin method chỉ khai báo interface nó CẦN.

IExecutionContext is a composition of 5 smaller interfaces: IDbContext, ICacheContext, IEventContext, IServiceContext, ILogger. A plugin method only declares the interface it NEEDS.

Vai tròRole

5 middleware chạy tuần tự mỗi request: LoadBalancer → TenantResolver → AuthGuard → RateLimiter → RouteMatcher → PermissionChecker. Mỗi middleware có quyền REJECT sớm — request không hợp lệ bị chặn ngay, không tốn resource L3/L4/L5. Total latency: ~3.9ms best case.

5 middleware run sequentially per request: LoadBalancer → TenantResolver → AuthGuard → RateLimiter → RouteMatcher → PermissionChecker. Each middleware has the right to REJECT early — invalid requests are blocked immediately, saving L3/L4/L5 resources. Total latency: ~3.9ms best case.

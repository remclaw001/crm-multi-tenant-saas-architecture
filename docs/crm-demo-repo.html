Mục đích
🎯 Thực hành
Codebase để áp dụng và kiểm chứng trực tiếp các quyết định kiến trúc được mô tả trong tài liệu này. Mỗi phase trong Build Roadmap có branch tương ứng để theo dõi tiến trình xây dựng từng bước.
🧪 Demo
Repo công khai dùng để minh hoạ các pattern multi-tenancy, plugin sandbox, async workers với code thực tế chạy được — không chỉ là diagram trên giấy.
📐 Nguyên tắc
Đây không phải production codebase. Ưu tiên rõ ràng, dễ đọc, và minh hoạ đúng pattern hơn là tối ưu hiệu năng. Comment giải thích lý do thiết kế (why), không chỉ what.
🔗 Quan hệ với docs
Repo này song song với tài liệu kiến trúc. Mỗi module trong backend/src/ có tài liệu giải thích tương ứng trong site này. Code và docs được cập nhật đồng thời.

Quick start
# Clone repo git clone https://github.com/remclaw001/crm-multi-tenant-saas-architecture cd crm-multi-tenant-saas-architecture/backend # Copy env và điền thông tin cp .env.example .env # Khởi động hạ tầng (PostgreSQL, Redis, RabbitMQ) docker compose up -d # Chạy migrations + seed data npm run db:migrate npm run db:seed # Khởi động dev server npm run start:dev # Chạy unit tests npm test

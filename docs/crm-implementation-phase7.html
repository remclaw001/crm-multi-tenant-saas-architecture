🔒 EncryptionService — AES-256-GCM PII Encryption
✓

Không cần external dependency. Node.js built-in crypto module đã có AES-256-GCM từ v10+. GCM (Galois/Counter Mode) cung cấp cả confidentiality (mã hóa) và integrity + authenticity (authentication tag) — tampered ciphertext sẽ throw khi decrypt, không corrupt silently.

encryption.service.ts — core logicTypeScript
const ALGORITHM = 'aes-256-gcm'; const IV_BYTES = 12; // 96-bit IV — NIST recommendation for GCM const TAG_BYTES = 16; // 128-bit authentication tag encrypt(plaintext: string): string { const iv = crypto.randomBytes(IV_BYTES); // ← fresh random IV mỗi lần encrypt const cipher = crypto.createCipheriv(ALGORITHM, this.key, iv); const encrypted = Buffer.concat([cipher.update(plaintext, 'utf8'), cipher.final()]); const authTag = cipher.getAuthTag(); // ← GCM integrity tag // Format: "iv_b64:authTag_b64:ciphertext_b64" return [iv, authTag, encrypted] .map(b => b.toString('base64')) .join(':'); } decrypt(ciphertext: string): string { const [ivB64, authTagB64, encB64] = ciphertext.split(':'); const decipher = crypto.createDecipheriv(ALGORITHM, this.key, iv); decipher.setAuthTag(authTag); // ← Throws ERR_CRYPTO_INVALID_TAG nếu ciphertext bị tamper return Buffer.concat([decipher.update(enc), decipher.final()]).toString('utf8'); }

🔑 PasswordService — bcrypt
✓

Tại sao cost 12? Cost factor quyết định 2^cost iterations. Cost 12 → 4096 iterations → ~250ms trên modern hardware (2024). Đủ chậm để attack dictionary cần nhiều năm; đủ nhanh để user không nhận thấy delay khi login.

password.service.tsTypeScript
const SALT_ROUNDS = 12; // Registration: hash plain password before storing async hash(password: string): Promise<string> { return bcrypt.hash(password, SALT_ROUNDS); // → "$2b$12$" stored in users.password_hash } // Login: compare without timing leak async verify(password: string, storedHash: string): Promise<boolean> { return bcrypt.compare(password, storedHash); // ← constant-time comparison inside bcrypt — không vulnerable timing attack }
ℹ

bcrypt vs Argon2 vs scrypt: bcrypt đủ tốt cho most production workloads. Argon2id là modern choice nhưng cần argon2 native addon. Với Node.js ecosystem hiện tại, bcrypt cost 12 là pragmatic choice.

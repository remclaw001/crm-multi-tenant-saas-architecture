Bản đồ L7 — 6 thành phần và data flowL7 Map — 6 Components & Data Flow

Nguyên tắc: L7 là tầng PASSIVE — không can thiệp vào business logic. Nó chỉ quan sát, thu thập, và cảnh báo. Mọi thành phần L7 nhận data từ các tầng khác nhưng không bao giờ thay đổi hành vi xử lý request.
Principle: L7 is a PASSIVE layer — it does not interfere with business logic. It only observes, collects, and alerts. Every L7 component receives data from other layers but never changes how requests are processed.

Audit Trail — Immutable Compliance Record

Audit Trail ≠ Logging. Logs phục vụ debugging, có thể xóa. Audit Trail phục vụ compliance/pháp lý, KHÔNG THỂ xóa/sửa. Logs ghi mọi thứ. Audit chỉ ghi data mutations.
Audit Trail ≠ Logging. Logs serve debugging and can be deleted. Audit Trail serves compliance/legal purposes and CANNOT be deleted or modified. Logs record everything. Audit only records data mutations.

│  WRITE PIPELINE — Async batched:                                          │
│                                                                              │
│  ┌────────────────────────────────────────────────────────────────────────┐ │
│  │                                                                        │ │
│  │  Plugin write xong                                                     │ │
│  │  → Audit Logger nhận event                                             │ │
│  │  → Push vào in-memory buffer (per instance)                            │ │
│  │  → Buffer flush mỗi 1 giây HOẶC khi buffer đầy (100 records)         │ │
│  │  → Batch INSERT INTO audit_log VALUES (...), (...), (...)              │ │
│  │  → Metadata DB (partitioned table)                                     │ │
│  │                                                                        │ │
│  │  Async batched → KHÔNG ảnh hưởng response latency.                   │ │
│  │  Buffer in-memory → nếu instance crash → mất ~1s audit data.         │ │
│  │  Acceptable trade-off cho performance. Logs vẫn có đầy đủ.           │ │
│  │                                                                        │ │

error handling at each stage
GATEWAY ERRORS (trước khi plugin code chạy)
─────────────────────────────────────────────────────────────────

  Tenant not found          → 404  Client sai URL
  Tenant suspended          → 403  Billing issue
  JWT invalid/expired       → 401  Re-login required
  Token tenant ≠ URL tenant → 403  Token theft attempt → alert
  Rate limit exceeded       → 429  + Retry-After header
  Route not found           → 404  Wrong endpoint

  → Nhanh, rẻ. Không tốn DB connection, không tạo context.
  → Vẫn log structured (request_id, tenant_id nếu có).


CONTEXT BUILD ERRORS
─────────────────────────────────────────────────────────────────

  Plugin not enabled        → 403  "Plugin Not Available"
  User lacks permission     → 403  "Forbidden"
  DB Pool exhausted         → 503  + Retry-After
                                   Circuit Breaker → OPEN
                                   Subsequent requests get fast-fail
                                   30s later → HALF-OPEN → try again

  → DB connection acquired? → must release before returning error.
  → try/finally ensures cleanup.


PLUGIN EXECUTION ERRORS
─────────────────────────────────────────────────────────────────

  ┌────────────────────────────────────────────────────────────┐
  │                                                            │
  │  Validation Error (BusinessError — permanent)               │
  │  Plugin throws: "Customer not found"                       │

  │  → If repeated → Circuit Breaker opens for DB              │
  │                                                            │
  │  Plugin Crash (PluginError — may be transient)              │
  │  Unhandled exception in plugin code                        │
  │  → Sandbox error boundary catches                          │
  │  → 500 Internal Server Error                               │
  │  → Log full stack trace with context                       │

  │                                                            │
  │  Timeout (PluginError)                                      │
  │  Plugin execution > 5000ms (premium) / 10000ms (VIP)       │
  │  → Sandbox kills execution                                 │
  │  → 504 Plugin Timeout                                      │
  │  → Log: which method, which queries, total time            │
  │                                                            │

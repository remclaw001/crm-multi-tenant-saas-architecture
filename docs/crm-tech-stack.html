Tech Stack Đề xuất

Lựa chọn công nghệ cho từng layer — mỗi thành phần có lựa chọn 1 (khuyến nghị) và lựa chọn 2 (thay thế khả thi).

Stack khuyến nghị

Frontend : React 18 + Next.js 15 + Module Federation # L1 Backend : Node.js 22 LTS + Fastify v5 + TypeScript 5 # L2–L3 Database : PostgreSQL 16 + Redis 7 # L5 Broker : RabbitMQ # L5 Search : Elasticsearch 8 # L5 Storage : MinIO # L5 Gateway : Kong # L2 Observ : Pino + OpenTelemetry + Prometheus + Grafana # L7 Deploy : Docker + Kubernetes + Terraform # DevOps

Nguyên tắc lựa chọn

Type safety xuyên suốt — TypeScript từ frontend đến backend, Zod cho runtime validation tại mọi system boundary.

Khớp kiến trúc đã thiết kế — Plugin sandbox dùng Isolated-VM; Query Interceptor hook vào Knex.js; cache key pattern t:<id>:<type>:<id> tường minh với ioredis.

Tách biệt mối lo ngại — Mỗi layer chỉ phụ thuộc interface, không phụ thuộc implementation trực tiếp (DIP).

Quan sát được từ ngày đầu — OpenTelemetry + Pino + Prometheus cấu hình ngay từ đầu, không bổ sung sau.
